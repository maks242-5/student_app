# StudentApp 🎓

Вебдодаток на Flask та SQLAlchemy для автоматизації навчального процесу, обліку студентів та моніторингу їхньої успішності.

## 🛠️ Функціонал додатка
* **Керування базою даних:** Реалізовано роботу з реляційними таблицями за допомогою ORM SQLAlchemy (користувачі, студенти, оцінки).
* **Авторизація клієнтів:** Безпечний вхід та реєстрація профілю користувача за унікальним нікнеймом.
* **Моніторинг успішності:** Зручний інтерфейс для перегляду списків студентів та керування їхніми навчальними даними.
* **Глобальна безпека:** Реалізовано надійне хешування паролів та захист усіх форм від кібератак за допомогою CSRF-токенів.

## 🚀 Стек технологій
* Python (Flask, Flask-SQLAlchemy, Flask-Login, Flask-WTF)
* Реляційна база даних (SQLite3/PostgreSQL)
* HTML5, CSS3 (чиста адаптивна верстка під ПК та мобільні пристрої)
